要在代码中使用 HarmonyOS 的沉浸光感（Immersive Light Effects），你需要根据使用的组件类型选择不同的配置方式。沉浸光感从 HarmonyOS 6.1.0（API 23）开始提供，它通过**沉浸式系统材质**和**空间动效**为组件带来通透、精致的视觉效果。<rsup>1</rsup>

## 一、开启沉浸光感

### 1. 应用级开启（推荐）
在 `module.json5` 中配置，对整个应用生效：
```json5
{
  "module": {
    "name": "entry",
    "type": "entry",
    "metadata": [{
      "name": "ohos.arkui.UIMaterial.state",
      "value": "enable" // 可选：default、enable、disable
    }]
  }
}
```

### 2. 组件级开启
#### 方式一：通过通用属性 `systemMaterial`
```typescript
import { uiMaterial } from '@kit.ArkUI';

@Entry
@Component
struct MyPage {
  build() {
    Column() {
      // 为 Column 组件开启沉浸光感
      Column() {
        Text('沉浸光感')
      }
      .width(328)
      .height(56)
      .borderRadius(28)
      .justifyContent(FlexAlign.Center)
      .systemMaterial(new uiMaterial.ImmersiveMaterial({
        style: uiMaterial.ImmersiveStyle.ULTRA_THIN, // 材质样式
        interactive: true, // 启用交互形变
        lightEffect: { color: undefined } // 光感交互反馈
      }))
    }
  }
}
```

#### 方式二：通过组件独有接口
适用于弹窗类组件：
```typescript
// Toast 示例
import { PromptAction, uiMaterial } from '@kit.ArkUI';

@Entry
@Component
struct MyPage {
  promptAction: PromptAction = this.getUIContext().getPromptAction();
  
  build() {
    Column() {
      Button('显示Toast')
        .onClick(() => {
          this.promptAction.showToast({
            message: '操作成功',
            duration: 2000,
            systemMaterial: new uiMaterial.ImmersiveMaterial({
              style: uiMaterial.ImmersiveStyle.THIN
            })
          });
        })
    }
  }
}
```

## 三、材质样式与级别

### 1. 材质样式（ImmersiveStyle）
| 样式 | 说明 | 适用场景 |
|------|------|----------|
| `ULTRA_THIN` | 超薄样式，透明度高 | 浮动工具栏、搜索框 |
| `THIN` | 薄样式，透明度较高 | 需要较强透明的场景 |
| `REGULAR` | 常规样式 | 通用场景 |
| `THICK` | 厚样式，模糊效果强 | 菜单、卡片 |
| `ULTRA_THICK` | 超厚样式，模糊很强 | 弹窗、半模态面板 |

### 2. 材质级别（MaterialLevel）
| 级别 | 说明 | 性能消耗 |
|------|------|----------|
| `ADAPTIVE` | 系统自适应（推荐） | 自动平衡 |
| `EXQUISITE` | 精致效果 | 高 |
| `GENTLE` | 柔和效果 | 中 |
| `SMOOTH` | 流畅效果 | 低 |

## 四、设备兼容性检查

建议在使用自定义材质前检查设备支持情况：

```typescript
import { hdsMaterial } from '@kit.UIDesignKit';
import { uiMaterial } from '@kit.ArkUI';

@Entry
@Component
struct MyPage {
  @State customMaterialLevel: hdsMaterial.MaterialLevel = hdsMaterial.MaterialLevel.EXQUISITE;
  
  aboutToAppear(): void {
    // 查询设备支持的材质类型
    let materialTypes: Array<hdsMaterial.MaterialType> = hdsMaterial.getSystemMaterialTypes();
    
    // 如果不支持 IMMERSIVE，降级到 SMOOTH
    if (materialTypes.indexOf(hdsMaterial.MaterialType.IMMERSIVE) < 0) {
      this.customMaterialLevel = hdsMaterial.MaterialLevel.SMOOTH;
    }
  }
  
  build() {
    Column() {
      // 使用检查后的材质级别
      Button('沉浸光感按钮')
        .systemMaterial(new uiMaterial.ImmersiveMaterial({
          style: uiMaterial.ImmersiveStyle.ULTRA_THIN,
          materialLevel: this.customMaterialLevel
        }))
    }
  }
}
```

## 六、注意事项

1. **版本要求**：需要 HarmonyOS 6.1.0（API 23）或更高版本
2. **性能优化**：低端设备建议使用 `ADAPTIVE` 或 `SMOOTH` 级别
3. **属性冲突**：设置 `systemMaterial` 后，避免同时设置背景色、阴影等样式
4. **深浅色适配**：沉浸光感会自动适配系统深浅色模式
5. **自动反色**：薄材质（`ULTRA_THIN`、`THIN`）可开启 `colorInvert: true` 确保文字可读性

## 七、最佳实践建议

1. **优先使用自适应模式**：`materialType: hdsMaterial.MaterialType.ADAPTIVE`
2. **焦点组件使用**：只在关键交互组件上使用，避免全屏滥用
3. **暗色模式效果更佳**：沉浸光感在深色背景上表现更好
4. **HDS组件优先**：对于导航栏和页签，优先使用 HDS 组件的 `systemMaterialEffect`<rsup>2</rsup><rsup>3</rsup>
5. **降级处理**：使用 `getSystemMaterialTypes()` 检查设备支持情况

通过以上配置，你可以轻松为应用添加高级的沉浸光感效果，提升用户体验和界面质感。

## ⚠️ 使用注意事项

1. **平台支持**：
   - UI Design Kit当前仅支持中国境内（港澳台除外）
   - 不同组件支持不同的设备类型，请查阅官方兼容性列表

2. **约束限制**：
   - HdsNavigation在横屏Stack模式下不支持合并工具栏到菜单栏
   - 标题栏默认采用层叠布局（位于内容区上层）
   - 图标批量处理接口最大并发数为10，单次最大处理量500个

3. **性能优化**：
   - 合理使用组件复用（@Reusable装饰器）
   - 对于长列表使用LazyForEach优化渲染性能
   - 避免在build()中执行耗时操作
